本程序用提取关键词的方式来分析一个微博用户所关心的事情，推测他的兴趣范围。

使用的分词是 JieBa 分词，利用其词性标注功能，主要提取指定用户 最近 200条微博中，高频率出现的名词。

为增强准备性，需过滤掉一些没有区分度的高频名词，可通过自定义词典meaninglessKeywordDict来进行这个过滤。
